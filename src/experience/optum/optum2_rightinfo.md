## Job Description

- Summer Project on Provider Fraud Detection using AI/ML
- Developed a Graph Database for storing
- Provider information under customized schema
- Wrote automated queries that ran social network analysis algorithms that measure the amount of risk in the graph
- Risk Factors are stored in Table that is reordered and weighted with Reinforced Learning through a built UI
- Built an Image Recognition Classifier that takes an Open Street View image through an address and classifies the potential risk that this address/provider is fraudulent. The method for this Image Classifier is currently pending on patent review.
